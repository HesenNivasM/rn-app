## Running the project

Clone the repo and cd into the directory

Create a .env file with the TMDB API Key:<br>
`echo API_KEY={API_KEY} >> .env`

Install dependencies:<br>
`npm i`

Run server:<br>
`npm run start`

Open Expo Go and scan the QR Code<br>
